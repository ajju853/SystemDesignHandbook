# Azure Overview

## Core Services

| Category | Service | Description |
|----------|---------|-------------|
| Compute | Virtual Machines | VMs |
| Compute | Azure Kubernetes Service (AKS) | Managed K8s |
| Compute | Azure Functions | Serverless |
| Compute | App Service | PaaS web apps |
| Storage | Blob Storage | Object storage |
| Storage | Azure SQL | Managed SQL Server |
| Database | Cosmos DB | Globally distributed NoSQL |
| Database | Azure Cache for Redis | Managed Redis |
| Networking | Virtual Network | VPC equivalent |
| AI | Azure OpenAI | GPT, DALL-E |

## Azure vs AWS

| Feature | Azure | AWS |
|---------|-------|-----|
| VM naming | Standard_D2s_v3 | t3.large |
| Kubernetes | AKS | EKS |
| Serverless | Functions | Lambda |
| NoSQL global DB | Cosmos DB | DynamoDB Global Tables |
| Managed SQL | Azure SQL | RDS |
| Enterprise | Office 365 integration | N/A |
