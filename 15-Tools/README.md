# 15 — Interactive Tools

Interactive web-based utilities for system design practice, estimation, and learning.

## Tools

| Tool | Description | Open |
|------|-------------|------|
| **Quiz Generator** | Generates random system design quiz questions by module and difficulty. Tracks score and shows explanations. | [`quiz-generator.html`](quiz-generator.html) |
| **Capacity Calculator** | Estimates traffic, storage, bandwidth, compute, and cloud costs based on DAU, request patterns, and data characteristics. | [`capacity-calculator.html`](capacity-calculator.html) |
| **Failure Simulator** | Visual architecture diagram with toggleable component failures. Simulates cascading failures and shows impact on uptime, data loss, and degraded services. | [`failure-simulator.html`](failure-simulator.html) |

## Usage

Open any `.html` file in a browser. No server or dependencies required.

## Future Tools

- Architecture Simulator: Drag-and-drop component wiring with live load testing
- Cost Optimizer: Recommends right-sizing, reserved instances, and spot usage
- Migration Planner: Step-by-step migration from monolith to microservices
