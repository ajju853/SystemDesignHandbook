# AWS Certifications

## Learning Paths

### Foundational
| Certification | Code | Duration | Cost | Target |
|--------------|------|----------|------|--------|
| Cloud Practitioner | CLF-C02 | 2-3 weeks | $100 | Non-technical, managers, sales |

### Associate Level
| Certification | Code | Duration | Cost | Target |
|--------------|------|----------|------|--------|
| Solutions Architect Associate | SAA-C03 | 8-12 weeks | $150 | Architects, developers |
| Developer Associate | DVA-C02 | 6-8 weeks | $150 | Developers |
| SysOps Administrator Associate | SOA-C02 | 6-8 weeks | $150 | Operations, sysadmins |

### Professional Level
| Certification | Code | Duration | Cost | Target |
|--------------|------|----------|------|--------|
| Solutions Architect Professional | SAP-C02 | 12-16 weeks | $300 | Senior architects |
| DevOps Engineer Professional | DOP-C02 | 12-16 weeks | $300 | DevOps engineers |

### Specialty
| Certification | Code | Duration | Cost |
|--------------|------|----------|------|
| Advanced Networking | ANS-C01 | 8-12 weeks | $300 |
| Security | SCS-C02 | 8-12 weeks | $300 |
| Machine Learning | MLS-C01 | 8-12 weeks | $300 |
| Data Analytics | DAS-C01 | 8-12 weeks | $300 |
| Database | DBS-C01 | 8-12 weeks | $300 |
| SAP on AWS | PAS-C01 | 8-12 weeks | $300 |

## Recommended Study Order
1. Cloud Practitioner → Solutions Architect Associate → Developer Associate
2. Solutions Architect Professional → DevOps Engineer Professional
3. Specialty (pick based on domain)
